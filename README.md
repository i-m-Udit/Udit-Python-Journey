# 🐍 My Python Learning Journey

Welcome to my Python progress repository! This is where I've documented my journey of learning Python from scratch — including chapter-wise exercises, practice snippets, and small mini-projects I built along the way.

## 📖 About This Repository

This repo is basically a diary of how I learned Python. I'm not a professional developer (yet!) — I started as a complete beginner and used this space to practice, experiment, and slowly build my understanding of programming concepts.

Most of the code here follows the **[Code With Harry Python Course](https://www.codewithharry.com/)**, which I used as my primary learning resource. The chapters and topics in this repository are structured based on that course, so you'll find folders/files corresponding to different lessons and concepts as I progressed through it.

> This is **past progress**, not polished production code. Expect beginner-level mistakes, messy code, and lots of trial and error — that's the whole point! It's a record of growth, not a portfolio of perfection.

## 🗂️ What's Inside

- **Chapter-wise practice files** — code written while following along with Code With Harry's Python tutorials (variables, loops, functions, OOP, file handling, etc.)
- **Mini projects** — small practice projects I built to apply what I learned (e.g. calculators, games, simple automation scripts, etc.)
- **Random experiments** — snippets where I was just messing around to understand how something works

## 🎯 Why I Made This

- To track my own progress and look back on how far I've come
- To have a reference for concepts I might forget later
- To keep myself motivated by seeing consistent practice over time
- To eventually build on this foundation and create more advanced projects
